# ODE Solver Visualizer (Python + Google Colab)

An interactive Ordinary Differential Equation (ODE) solver and visualizer built using Python. This project uses SciPy for numerical solving and Matplotlib for real-time visualization with interactive sliders.

## 🚀 Features
- Solve first-order ODEs numerically
- Interactive parameter control (ipywidgets)
- Real-time graph visualization
- Logistic growth model included
- Easy to run in Google Colab

## 🧠 Example Equations
- Exponential decay/growth: dy/dt = a·y  
- Logistic growth: dy/dt = r·y(1 - y/K)

## 🛠️ Tech Stack
- Python
- NumPy
- SciPy
- Matplotlib
- ipywidgets

## ▶️ How to Run
1. Open Google Colab
2. Paste the notebook cells
3. Run all cells
4. Use sliders to change parameters and see live results
